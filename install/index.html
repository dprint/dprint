Install dprint

Install using one of the methods below.

  • Shell (Mac, Linux, WSL):

    curl -fsSL https://dprint.dev/install.sh | sh

  • Powershell (Windows):

    iwr https://dprint.dev/install.ps1 -useb | iex

  • Scoop (Windows):

    scoop install dprint

  • Homebrew (Mac):

    brew install dprint

  • Cargo (builds and installs the cargo package from source):

    # this will be slower since it builds from the source
cargo install --locked dprint

  • npm:

    # for your project
npm install dprint
npx dprint help

# or install globally
npm install -g dprint
dprint help

  • python/uv via https://github.com/trim21/dprint-py:

    uv add dprint-py
uv run dprint help

  • asdf-vm (asdf-dprint):

    asdf plugin-add dprint https://github.com/asdf-community/asdf-dprint
asdf install dprint latest

  • Arch Linux:

    Install with any AUR helper, for example:

    paru -S dprint

    or binaries

    paru -S dprint-bin

For binaries and source, see the GitHub releases.

Editor Extensions

  • Visual Studio Code
  • IntelliJ - Thanks to the developers at Canva
  • Neovim with nvim-lspconfig
  • The dprint lsp subcommand provides code formatting over the language server protocol. This can be used to format in other editors.
